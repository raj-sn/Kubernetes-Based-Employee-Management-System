# Kubernetes-Based Employee Management System

## Project Overview

This project demonstrates deploying a Flask-based Employee Management System on Kubernetes using Minikube running on AWS EC2.

The application is containerized using Docker, stored in Docker Hub, and deployed using Kubernetes Deployments and Services.

## Kubernetes Components

### Deployment

Manages and maintains application pods.

### Service

Provides network access to pods.

### Secret

Stores database credentials securely.

### Pod

Smallest deployable Kubernetes unit.

---

## Demonstrations

### Self-Healing

Deleting a pod automatically creates a new replacement pod.

### Scaling

Application replicas can be increased from 5 to 7 pods.
